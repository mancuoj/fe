# [104. 二叉树的最大深度](https://leetcode.cn/problems/maximum-depth-of-binary-tree/) <Badge type="tip" text="Easy" />

## Talk

- 遍历
  - 二叉树的遍历框架，在前中后序上添加操作
  - 计算达到每一个根节点的高度，进入 ++，离开 --
- 子树推导
  - 递归计算左右子树的最大深度，取最大值 +1（根节点）


## Code

:::code-group

```ts [TS 版遍历框架]
function traverse(root: TreeNode | null) {
  if (root === null)
    return

  traverse(root.left)
  traverse(root.right)
}
```
