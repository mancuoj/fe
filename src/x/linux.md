## 现代命令行工具

| 工具                          | 说明（替代）      |
| ----------------------------- | ----------------- |
| zsh + oh-my-zsh + zsh-plugins | bash              |
| fzf                           | fuzzy finder      |
| neofetch                      | system info tool  |
| tealdeer(tldr)                | cheatsheets       |
| bat                           | cat               |
| exa                           | ls                |
| slides                        | presentation tool |
| gdu                           | du                |
| duf                           | df                |
| bottom(btm)                   | top               |
